# MiniProj2
Programação Web Avançada 2023 | Tarefa 3.2: Miniprojeto - Desenvolvimento do front-end

A tarefa 3.2 corresponde ao desenvolvimento do front-end que visa complementar o projeto apresentado no livro recomendado, Projeto Animalec. O primeiro passo é implementar o projeto já disponibilizado pelo livro, num servidor local.

Fases da tarefa:

1.) Cada aluno deve ler os recursos partilhados para consolidar os conhecimentos JavaScript, e compreender a estrutura do código partilhado, o conjunto de funcionalidades relacionadas com a iteração do utilizador. Para esta fase da tarefa estar terminada considerada estar terminada deverá ter o font-end totalmente funcional.

2.) Cada aluno deve implementar as novas entidades planeadas na tarefa 3.1, com todas as funcionalidades previstas.  

3.) Cada aluno deve explorar a API que será a base de pedidos para gerir a informação disponibilizada: http://fcawebbook2.herokuapp.com/, bem como a comunicação com o servidor Web através de API nativas e bibliotecas, que tipicamente são baseados em JSON.

4.) Todos os ficheiros desenvolvidos devem ser submetidos na plataforma Git, na conta individual de cada aluno. O projeto no Git deve-se chamar de MiniProj2. A avaliação do projeto será realizada a partir dos ficheiros existentes no Git.
